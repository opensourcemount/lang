# Український переклад OPNsense

Цей файл містить початковий український переклад для OPNsense.

## Статус перекладу

- **Загальна кількість рядків**: 10,164
- **Перекладено рядків**: 468
- **Відсоток перекладу**: 4.60%

## Що перекладено

Переклад включає широкий спектр термінів інтерфейсу та системи:

### Основні дії
- Add → Додати
- Save → Зберегти  
- Cancel → Скасувати
- Delete → Видалити
- Edit → Редагувати
- Enable → Увімкнути
- Disable → Вимкнути
- Apply → Застосувати

### Статуси
- Active → Активний
- Enabled → Увімкнено
- Running → Працює
- Connected → Підключено
- Online → В мережі

### Загальні терміни
- Name → Назва
- Description → Опис
- Address → Адреса
- Network → Мережа
- Firewall → Брандмауер
- System → Система
- Settings → Налаштування

### Розширені терміни
- Certificate → Сертифікат
- Authentication → Автентифікація
- Configuration → Конфігурація
- Interface → Інтерфейс
- Protocol → Протокол
- Security → Безпека
- Monitoring → Моніторинг

### Технічні терміни
- Hardware → Апаратне забезпечення
- Software → Програмне забезпечення
- Database → База даних
- Encryption → Шифрування
- Debugging → Налагодження
- Performance → Продуктивність

## Як продовжити переклад

1. **Використовуйте POEditor**: https://translate.opnsense.org/
2. **Локальний переклад**: Редагуйте файл `uk_UA.po`
3. **Автоматичні скрипти перекладу**:
   - `translate_basic.py` - базові терміни
   - `translate_extended.py` - розширені терміни
   - `translate_short_phrases.py` - короткі фрази
   - `translate_additional_words.py` - додаткові слова

## Правила перекладу

Згідно з офіційними рекомендаціями OPNsense:

- Зберігайте оригінальну пунктуацію та пробіли
- Форми типу "interface(s)" перекладайте як множину без дужок
- Для динамічних аргументів використовуйте нумерацію `%1$s`, `%2$s`
- Повідомляйте про помилки в оригінальних рядках через коментарі
- Повідомляйте про HTML теги в рядках

## Наступні кроки

1. Зв'язатися з командою OPNsense для додавання української мови
2. Продовжити переклад через POEditor
3. Перекласти найважливіші розділи: меню, повідомлення про помилки, налаштування
