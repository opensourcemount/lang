# Український переклад OPNsense

Цей файл містить початковий український переклад для OPNsense.

## Статус перекладу

- **Загальна кількість рядків**: 12,325
- **Перекладено рядків**: 155
- **Відсоток перекладу**: 1.26%

## Що перекладено

Переклад включає найважливіші базові терміни інтерфейсу:

### Основні дії
- Add → Додати
- Save → Зберегти  
- Cancel → Скасувати
- Delete → Видалити
- Edit → Редагувати
- Enable → Увімкнути
- Disable → Вимкнути
- Apply → Застосувати

### Статуси
- Active → Активний
- Enabled → Увімкнено
- Running → Працює
- Connected → Підключено
- Online → В мережі

### Загальні терміни
- Name → Назва
- Description → Опис
- Address → Адреса
- Network → Мережа
- Firewall → Брандмауер
- System → Система
- Settings → Налаштування

## Як продовжити переклад

1. **Використовуйте POEditor**: https://translate.opnsense.org/
2. **Локальний переклад**: Редагуйте файл `uk_UA.po`
3. **Автоматичний переклад**: Використовуйте скрипт `translate_basic.py`

## Правила перекладу

Згідно з офіційними рекомендаціями OPNsense:

- Зберігайте оригінальну пунктуацію та пробіли
- Форми типу "interface(s)" перекладайте як множину без дужок
- Для динамічних аргументів використовуйте нумерацію `%1$s`, `%2$s`
- Повідомляйте про помилки в оригінальних рядках через коментарі
- Повідомляйте про HTML теги в рядках

## Наступні кроки

1. Зв'язатися з командою OPNsense для додавання української мови
2. Продовжити переклад через POEditor
3. Перекласти найважливіші розділи: меню, повідомлення про помилки, налаштування
